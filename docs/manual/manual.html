CCI請求書・ラベル発行システム
使い方マニュアル

対象バージョン: v2.1.10 ／ 最終更新: 2026年6月

このマニュアルについて

  • 商工会議所向けの請求書・領収書・宛名ラベル発行アプリの操作ガイドです。
  • 各機能はウィンドウ上部のタブで切り替えます。
  • データはアプリと同じフォルダに自動保存されます（バックアップ推奨）。

概要

このアプリは、商工会議所が会員向けに発行する請求書・領収書・宛名ラベルをまとめて管理・印刷できるシステムです。

タブ名主な用途
単発発行1件単位で請求書・領収書を手入力して発行
まとめて発行案件（プロジェクト）単位で一括発行・入金管理
宛名ラベル発行会員リストから宛名ラベル・名札を印刷
修正・再発行発行済み書類の内容修正・PDF再出力
登録・マスタスタッフ・業務名・請求テンプレート・会員情報の管理
設定発行元情報・メール・バックアップ・操作ログ

インストール・起動

初回インストール

  1. GitHub のリリースページから CCIBillingLabel_Setup_x.x.x.exe をダウンロードします。
  2. ダウンロードしたファイルをダブルクリックして実行します。
  3. 画面の指示に従ってインストールを完了します（管理者権限は不要です）。
  4. デスクトップまたはスタートメニューの「CCI請求書・ラベル発行システム」を起動します。
インストール先はデフォルトで %LOCALAPPDATA%\CCIBillingLabel になります。変更は不要です。

起動時の動作

  • 起動直後に自動バックアップが実行されます（ステータスバーに完了メッセージが表示されます）。
  • 新しいバージョンがある場合、画面上部に黄色い通知バーが表示されます（バージョンアップ参照）。

単発発行

1件ずつ手入力して請求書または領収書を発行します。宛先や金額が毎回異なる場合に使います。

請求書の発行

  1. 「単発発行」タブ →「請求書」サブタブを開きます。
  2. 宛先・件名・担当スタッフ・請求項目を入力します。
  3. 「PDF生成・発行」ボタンをクリックします。
  4. PDFが自動的に開きます。印刷してご使用ください。

主な入力項目

項目説明
宛先会員の事業所名・氏名
件名案件名（例：令和6年度 年会費）
発行日デフォルトは本日の日付
担当スタッフスタッフ管理で登録済みのスタッフを選択
請求項目項目名・単価・数量・税区分（テンプレートから選択可能）
備考振込先・支払期限など（任意）
よく使う項目は「登録・マスタ」→「請求項目テンプレート」に登録しておくと、ワンクリックで入力できます。

領収書の発行

  1. 「単発発行」タブ →「領収書」サブタブを開きます。
  2. 宛先・金額・担当者・但し書きを入力します。
  3. 「PDF生成・発行」ボタンをクリックします。

まとめて発行

案件（プロジェクト）単位で会員を一括管理し、請求書・領収書をまとめて発行します。年会費・講座料など同じ案件で複数名に発行する場合に使います。

① データ作成

  1. 「まとめて発行」タブ →「データ作成」サブタブを開きます。
  2. 「＋ 新規案件」ボタンで案件を作成します（年度・業務区分・件名・発行元を設定）。
  3. 会員マスタからメンバーを追加するか、会員リストをインポートします。
  4. 各メンバーの金額・請求項目を確認・調整します。

② 請求書発行

  1. 「請求書発行」サブタブを開きます。
  2. 発行する案件と対象メンバーを選択します。
  3. 「一括PDF生成」ボタンをクリックします。
  4. 全メンバー分の請求書PDFがまとめて生成・開かれます。
メール設定が完了している場合、発行ボタンをクリックすると1件ずつM365メール送信の確認ダイアログが表示されます。「送信する」「スキップ」「すべてスキップ」から選択できます。

③ 領収書発行

  1. 「領収書発行」サブタブを開きます。
  2. 入金済みのメンバーを選択して「領収書発行」をクリックします。

④ 入金管理

  1. 「入金管理」サブタブを開きます。
  2. 入金があったメンバーの行を選択し、「入金記録」ボタンで入金日・金額を登録します。
  3. 状態が「入金済み」に変わります。領収書が発行できるようになります。

宛名ラベル発行

案件の会員リストから宛名ラベルや名札を印刷します。市販のラベル用紙（A-ONE 28185・28187・51002）に対応しています。

① 絞り込みと件名選択

項目説明
年度「すべての年度」を選ぶと全案件を表示。特定年度に絞ることも可能。
業務区分業務の種類でフィルタ。「すべて」でフィルタなし。
件名案件を選択するとメンバー一覧が表示されます。
検索事業所名・代表者名でリアルタイム絞り込み。

セルの編集

  • 編集可能なセルをダブルクリック（または選択してクリック）すると編集モードになります。
  • Enter で確定、Tab で次のセルへ移動します。
  • 長い事業所名・役職名は Alt + Enter で折り返し（改行）を入力できます。テーブル上では「 ｜ 」で表示され、PDF には改行で印刷されます。

ヘッダークリックでソート

列ヘッダーをクリックすると昇順（▲）・降順（▼）でソートできます。ソート後もチェック状態は保持されます。

③ モード・用紙・PDF生成

モードの選択

モード内容用途

用紙の選択

用紙型番サイズ配置モード

「名札」モードは A-ONE 51002 固定です。宛名・事業所名のみモードでは 28185（3列）と 28187（2列）を選択できます。

PDF生成の手順

  1. モード・用紙・フォントを選択します。
  2. 対象メンバーのチェックボックスをオンにします（ヘッダーの☑で全選択可）。
  3. 「ラベルPDF生成」ボタンをクリックします。
  4. PDFが自動的に開きます。プリンターで印刷してください。
ステータスバーに「X 件表示 ／ チェック済み Y 件」と表示されます。生成前に件数を確認してください。
セルを編集した内容（折り返し・住所修正等）はデータベースには保存されません。PDF生成のみに反映されます。

修正・再発行

発行済みの請求書・領収書の内容を修正したり、PDFを再出力したりします。

書類の検索

絞り込み項目説明
年度発行年度で絞り込み
業務区分業務の種類で絞り込み
件名案件名で絞り込み
種別請求書 / 領収書 / すべて
検索ボックス宛先・件名のキーワード検索

内容修正

  1. 修正したい書類の行を選択します。
  2. 「内容修正」ボタンをクリックします（「発行済み」状態の書類のみ有効）。
  3. 修正ダイアログで内容を変更し、保存します。
内容修正は「発行済み」ステータスの書類のみ可能です。領収書には内容修正はありません（再発行のみ）。

再発行（PDF再出力・メール送付）

発行方法の選択

「発行方法」コンボボックスで 印刷 または メール送付 を選択してから再発行ボタンをクリックします。

発行方法動作
印刷保存先を指定してPDFを出力します
メール送付M365メールで宛先にPDFを送信します
  1. 再発行したい書類の行を選択します。
  2. 「発行方法」を選択します。
  3. 「再発行（PDF再出力）」ボタンをクリックします。
  4. 印刷の場合は保存先を指定、メール送付の場合は確認ダイアログで内容を確認して送信します。
再発行した操作は操作ログに記録されます。

登録・マスタ

システムで使用する各種マスタデータを管理します。最初に設定しておくと、発行作業がスムーズになります。

スタッフ管理

書類に印字する担当者（職員）と、発行通知の送信先となる所属長を管理します。

所属長

M365メールで発行通知を受け取る所属長を登録します。

列説明
ID自動付番
氏名所属長の名前（必須）
メール発行通知メールの送信先アドレス

「追加」「編集」「削除」ボタンで管理します。

職員

書類に印字する担当者を登録します。

列説明
氏名書類に印字される担当者名
担当所属長この職員の発行通知先となる所属長
所属長フラグこの職員自身が所属長である場合にチェック
メールアドレス職員本人のメールアドレス
管理者管理者権限の有無
状態有効 / 無効
  • 退職等で使わなくなった職員は「無効化」できます（削除ではないので履歴は残ります）。
  • 「パスワードをリセット」で初期パスワードに戻せます（管理者のみ）。
  • 「管理者に設定 / 解除」で管理者権限を切り替えられます。

業務名（業務区分）

請求書・領収書・案件を分類する業務区分を登録します。

  • 例：「年会費」「研修費」「会場費」など
  • 一度登録した業務名は変更・削除できます。

請求項目テンプレート

よく使う請求項目をテンプレートとして登録します。

テンプレートの登録

  1. 「＋ 新規テンプレート」ボタンをクリックします。
  2. 各項目を入力して「登録」をクリックします。

入力項目

項目説明
業務名登録済みの業務区分から選択（必須）
項目名・但し書き請求書に印字されるテキスト（必須）
単価0〜9,999,999円
単位既定値「式」
税区分消費税10% / 消費税8% / 非課税 / 不課税
テンプレートは「単発発行」「まとめて発行」の請求項目入力画面から呼び出せます。登録しておくと手入力の手間が省けます。

会員マスタ

会員の住所録（マスタデータ）を管理します。個別の追加・編集・削除と、CSVからの一括インポートが利用できます。

一覧表示と検索

  • 検索バーに入力すると、事業所名・フリガナ・氏名・会員番号でリアルタイム絞り込みができます。
  • 列ヘッダーをクリックすると昇順（▲）・降順（▼）でソートできます。
  • 右上に「表示：X 件 / 全 Y 件」の件数が表示されます。

CSVインポート（一括登録）

CSVインポートを実行すると既存の会員データがすべて削除されてから再登録されます。実行前にバックアップを取ってください。
  1. 「ファイルを選択…」でCSVファイルを指定します。
  2. 「インポート実行」をクリックすると列マッピングダイアログが開きます。
  3. CSVの各列をどのフィールドに取り込むか確認・変更して「インポート実行」をクリックします。

対応ヘッダー例：会員番号・事業所名・フリガナ・氏名・電話番号・メール・郵便番号・住所

設定

発行元情報

請求書・領収書のヘッダーに印字される商工会議所の情報を管理します。複数の発行元を登録し、案件ごとに切り替えて使用できます。

発行元一覧

  • 「＋ 発行元追加」で新規発行元を登録します。
  • 「★ デフォルトに設定」で選択した発行元を既定の発行元にします。
  • 「編集」「削除」で発行元情報を変更・削除できます。
項目説明
商工会議所名書類ヘッダーに大きく印字
代表者名代表者として印字
郵便番号・住所発行元住所
電話番号・FAX連絡先
ロゴ画像書類に印字するロゴ（PNG/JPG）

銀行口座

選択中の発行元に紐づく振込先口座を複数登録できます。請求書の振込先として使用します。

印鑑画像

電子印鑑の画像（PNG/JPG）を登録します。画像はデータベースに保存されるため、複数PC間で共有されます。

発行元情報は書類発行時に参照されます。変更した場合は既発行書類の再発行が必要です。

メール設定

請求書・領収書・督促メールの送信と発行通知の設定です。Microsoft 365（Graph API）を使用します。

Microsoft 365 メール送信（Graph API）

項目説明
アプリケーション（クライアント）IDMicrosoft Entra ID のアプリ登録で取得
ディレクトリ（テナント）ID組織のテナント ID
Microsoft Entra ID でアプリ登録（Public Client）と Mail.Send 権限が必要です。詳しくは「サーバー設定マニュアル」を参照してください。

送信メールテンプレート

請求書・領収書・督促（支払期限超過）の3種類について、件名・本文をカスタマイズできます。

本文には差し込みタグ（{宛先}・{担当者名}・{件名}・{金額}・{発行番号}・{支払期限} など）を使用でき、送信時に各宛先の情報に置き換えられます。

発行通知の送信先

書類発行時に担当の所属長へ通知メールが送信されます。送信先は「登録・マスタ」→「スタッフ管理」の所属長メールアドレスで職員ごとに設定します。

バックアップ

データベースの自動バックアップを設定します。

  • 起動時に自動でバックアップが作成されます（ステータスバーに完了メッセージが表示されます）。
  • バックアップ先フォルダ・世代数を設定できます。
  • 手動バックアップも実行できます。

操作ログ

誰が・いつ・何をしたかの履歴を確認できます。

絞り込み

項目説明
開始日・終了日期間を指定（既定: 過去1ヶ月）
操作フィルタ発行 / 内容修正 / 再発行 / 入金記録 / メール送信 など
検索ボックス担当者・操作・詳細でキーワード検索

CSV出力

「CSV出力」ボタンで表示中のログをCSVファイルとして保存できます。監査・報告書作成にご活用ください。

管理者機能

管理者機能のボタンはデータを削除します。操作前に必ずバックアップを取ってください。元に戻すことはできません。
機能削除されるデータ残るデータ

管理者機能は管理者権限を持つスタッフでログインしている場合にのみ表示されます。

バージョンアップ

新しいバージョンがリリースされると、アプリ起動時に画面上部に黄色い通知バーが表示されます。

  1. 通知バーの「ダウンロード」ボタンをクリックします。インストーラーが自動的にダウンロードされます。
  2. ダウンロードが完了すると「今すぐ更新して再起動」ボタンが表示されます。
  3. ボタンをクリックするとインストーラーが起動し、アプリが自動的に終了します。
  4. インストーラーの指示に従って更新を完了します。
更新中にデータが失われることはありません。ただし、念のため更新前にバックアップを取ることをお勧めします。

https://github.com/mozu93/cci-billing-label/releases

既存のアプリの上から上書きインストールできます。データは保持されます。
